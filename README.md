#Introduction

Script to segment and quantify beads in microscopy images.

#Installation

gem install find_beads

#Running

To list command line options for processing:

find_beads --help

To run:

find_beads [args]

#Documentation

API documentation is available [here](http://rubydoc.info/gems/find_beads).
